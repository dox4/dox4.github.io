JVM 的运行时数据区包含以下 5 个：
1. 方法区
2. 虚拟机栈
3. 本地方法栈
4. 堆
5. 程序计数器

除了最后一个**程序计数器**，其他四个内存区域都可能发生 `OutOfMemory` 异常。程序计数器对应的就是 `CPU` 里的 `PC` 寄存器，功能和名字都一样，这个地方理论上是不会发生内存溢出的。
